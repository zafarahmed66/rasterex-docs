Callback event that is triggered when a pan operation is completed.

###Callback Parameters

- **X**: Offset left.
- **Y**: Offset top.
- **Pagerect**: Object containing the dimensions and position of the page.

### JavaScript

```javascript
var pagerect = {
    x: // upper left position of page,
    y: // upper left position of page,
    w: // width of page,
    h: // height of page,
};
```
