Sets the line style used for markup. **Note**: Not all browsers currently support this feature.

### Syntax

```typescript
RxCore.setLineStyle(linestyle)
```

### Parameters

- `linestyle`: **number** — Line style option:
  - `0`: solid
  - `1`: dashed
  - `2`: dotted
  - `3`: dash dot dot dash
  - `4`: dash dot

### Returns

- **NA** — This method does not return a value.
